Windows API Code Pack Tests
=============================================================

The tests for the Windows API Code Pack are implemented using 
xUnit (http://xunit.codeplex.com).

To execute the tests, follow the steps below:

1. Build all tests
2. Run $\Tools\xunit-1.5\xunit.gui.exe
3. Load Tests.xunit and execute all tests

For more information on xUnit, visit: http://xunit.codeplex.com/
For  a Visual Studio Add-In, visit: http://www.testdriven.net/
